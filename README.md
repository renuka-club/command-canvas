# Command Canvas - Admin Dashboard

## Project Info

This is a customized admin dashboard built with React and Shadcn UI.

## Getting Started

### Prerequisites
- Node.js & npm installed

### Installation

1. Clone the repository
```sh
git clone <YOUR_GIT_URL>
```

2. Navigate to project directory
```sh
cd command-canvas
```

3. Install dependencies
```sh
npm install
```

4. Start development server
```sh
npm run dev
```

## Technologies
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Recharts
- React Query
